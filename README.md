# 🕷️ gameKis - Front-End Heroes Quiz Platform

Uma plataforma web completa de quiz gamificado sobre programação Front-End, com visual inspirado no estilo heróico (cores vibrantes, vermelho, azul, efeitos neon e animações dinâmicas).

## 🚀 Como Rodar o Projeto

Siga os passos abaixo para iniciar a jornada:

### 1. Clonar ou Baixar o Projeto
Certifique-se de que você está na pasta raiz do projeto `gameKis`.

### 2. Iniciar o Backend
O backend armazena as perguntas e os recordes dos jogadores em arquivos JSON.

```bash
cd backend
npm install
npm start
```
*O servidor rodará em `http://localhost:3001`*

### 3. Iniciar o Frontend
O frontend é construído com React, TypeScript e Vite.

```bash
cd frontend
npm install
npm run dev
```
*O site abrirá em `http://localhost:5173`*

## 🎮 Funcionalidades

- **3 Níveis de Dificuldade:** Iniciante, Intermediário e Master.
- **50 Perguntas:** HTML, CSS, JS, React, Responsividade, Acessibilidade e Performance.
- **Sistema de Progressão:** Você precisa de pelo menos 70% de acerto para desbloquear a próxima fase.
- **Insígnias Dinâmicas:** Ganhe insígnias exclusivas ao completar cada nível.
- **Compartilhamento:** Compartilhe sua conquista diretamente no LinkedIn.
- **Ranking Global:** Veja quem são os melhores heróis do front-end.

## 🛠️ Stack Tecnológica

- **Frontend:** React + Vite + TypeScript + CSS Puro
- **Backend:** Node.js + Express
- **Banco de Dados:** JSON Local (NoSQL Style)
- **Ícones:** Lucide React
- **Efeitos:** Canvas Confetti
